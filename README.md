# Bannerbear

Ruby wrapper for [Bannerbear](https://www.bannerbear.com) - an image and video generation service.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bannerbear'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install bannerbear
